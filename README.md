# tiptap-tutorial

## 手順

### 1. ディレクトリの作成

```
mkdir tiptap-tutorial
```

### 2. プロジェクトの作成

```
npx create-remix@latest .
```

Initialize a new git repository? (recommended)  
Yes  

Install dependencies with npm? (recommended)
Yes
